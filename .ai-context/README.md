# AI Context for StreamPicks Website

This folder exists to give future AI coding agents durable context about the StreamPicks website, product direction, and implementation priorities.

Any coding agent working on this repo should read these files first:

1. product-vision.md
2. website-role.md
3. phase-1-website-refresh.md
4. technical-context.md
5. constraints-and-non-goals.md
6. copy-guidelines.md

## Purpose

The goal is to prevent future AI coding sessions from making good technical changes with bad product assumptions.

The website is part of the StreamPicks product system, but it is not intended to become a full browser version of the game in the near term.

## Priority order

When in doubt, optimize for:
1. clear product understanding
2. stronger emotional positioning
3. app download / sign-up conversion
4. future compatibility with account-linked and shared-content pages
5. preserving working site quality
