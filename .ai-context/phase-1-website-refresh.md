# Phase 1 Website Refresh

## Goal

Refresh the public website so it better communicates the product and creates stronger excitement and conversion without overbuilding the web experience.

## Main outcomes wanted

Phase 1 should improve:
- homepage clarity
- product positioning
- emotional appeal
- CTA structure
- premium framing
- feature hierarchy
- social proof and trust cues
- mobile readability and polish

## Messaging priorities

The site should more clearly communicate:
- this is a creator competition game
- daily reveals matter
- rank progression matters
- rivalry and social elements matter
- StreamPicks+ is a meaningful upgrade

## Section-level direction

### Hero
The hero should explain the product quickly and make it feel exciting.
It should lean into:
- creator competition
- picks / drafts / rivalries
- daily reveals
- rank climbing
- social flexing

### How it works
This should become simple and visual.
A user should understand the basic flow in seconds.

### Why people come back
Add or strengthen a section explaining why the game is sticky:
- daily reveals
- rivalry
- rank movement
- teams
- creator momentum

### StreamPicks+
The premium section should be framed around:
- power
- strategy
- prestige
- identity
- deeper control

Avoid making it feel like it only offers “more slots.”

### Social proof
The site should feel like a living product with a community, not just a static pitch page.

### CTAs
Improve clarity and placement of:
- app download / join
- reserve username / join early
- premium curiosity / learn more

## Out of scope for Phase 1

Do not build:
- full gameplay on web
- full account dashboard
- full auth experience
- full leaderboards
- deep share system implementation
- major backend features
